# 📌 Project Overview

House Price Prediction is a supervised Machine Learning regression project that predicts the median value of houses based on different housing features such as income, house age, average rooms, population, and geographical location.

The project demonstrates a complete machine learning workflow including data exploration, preprocessing, normalization, model training, prediction, and performance evaluation.

---

# 🎯 Objectives

* Predict house prices using Machine Learning.
* Understand regression techniques.
* Perform data preprocessing and normalization.
* Train a Linear Regression model.
* Evaluate model performance using regression metrics.

---

# 📂 Dataset

**Dataset:** California Housing Dataset

The dataset is provided by **Scikit-Learn**.

### Features

* Median Income
* House Age
* Average Rooms
* Average Bedrooms
* Population
* Average Occupancy
* Latitude
* Longitude

### Target

Median House Value

### Dataset Statistics

| Item          | Value              |
| ------------- | ------------------ |
| Total Samples | 20,640             |
| Features      | 8                  |
| Target        | Median House Value |

Dataset Source:

https://scikit-learn.org/stable/modules/generated/sklearn.datasets.fetch_california_housing.html
